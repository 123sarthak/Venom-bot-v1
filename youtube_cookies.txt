# YouTube Cookies Instructions
# 
# To fix "sign in to confirm you're not a bot" error:
# 
# 1. Go to YouTube.com in your browser
# 2. Make sure you're logged in to your Google account
# 3. Open Developer Tools (F12)
# 4. Go to Application/Storage tab
# 5. Find Cookies under youtube.com
# 6. Copy these cookie values:
#    - SID
#    - HSID
#    - SSID
#    - APISID
#    - SAPISID
#    - __Secure-3PAPISID
#    - __Secure-3PSID
#    - __Secure-1PAPISID
#    - __Secure-1PSID
#    - __Secure-3PSIDCC
#    - __Secure-1PSIDCC
# 
# 7. Paste them below in this format (remove the # at the start):
#    SID=your_sid_value; HSID=your_hsid_value; SSID=your_ssid_value; ...
# 
# Example:
# SID=abc123; HSID=def456; SSID=ghi789; APISID=jkl012; SAPISID=mno345; __Secure-3PAPISID=pqr678; __Secure-3PSID=stu901; __Secure-1PAPISID=vwx234; __Secure-1PSID=yz5678; __Secure-3PSIDCC=abc901; __Secure-1PSIDCC=def234
# 
# Leave this file empty if you don't have cookies yet.
# The bot will work without cookies, but may have limitations.
